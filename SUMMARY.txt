================================================================================
✨ KEY FEATURES & ALGORITHMS
================================================================================

ATS SCORING ALGORITHM:
  - 40% Skill Matching (keyword frequency)
  - 40% TF-IDF Similarity (scikit-learn cosine similarity)
  - 20% Formatting Bonus (well-structured PDF)
  - Range: 0-100 score
  - Strength Tiers: Excellent (80-100), Strong (65-79),
                    Moderate (50-64), Weak (0-49)

SKILL EXTRACTION:
  - Pattern matching with regex word boundaries
  - 150+ predefined skills database
  - Case-insensitive matching
  - Automatic standardization

GAP ANALYSIS:
  - Set operations (intersection, difference, union)
  - Overlap percentage calculation
  - Contextual recommendation generation
  - Priority-based sorting

ROADMAP GENERATION:
  - Rule-based path matching
  - 6 predefined career paths
  - 3-level progression (Beginner → Advanced)
  - Duration estimates per module
  - Flexible pacing (52-78 weeks typical)

================================================================================
🚀 QUICK START COMMANDS
================================================================================

AUTOMATIC START (One-liner):
  Windows: start-all.bat
  macOS/Linux: ./start-all.sh

MANUAL START (3 Terminals):

Terminal 1 - Backend:
  cd server
  npm install
  npm run dev

Terminal 2 - ML Service:
  cd ml-service
  python -m venv venv
  venv\Scripts\activate  (Windows) or source venv/bin/activate
  pip install -r requirements.txt
  python main.py

Terminal 3 - Frontend:
  cd client
  npm install
  npm start

VERIFY SERVICES:
  curl http://localhost:5000/api/health
  curl http://localhost:8000/health
  Open http://localhost:3000 in browser

================================================================================
📊 API ENDPOINTS
================================================================================

POST /api/ats/upload
  Request:  multipart/form-data with PDF file
  Response: {score, matched[], missing[], strength}

POST /api/skill-gap/analyze
  Request:  {resume_text, job_description}
  Response: {present[], required[], missing[], recommendations[]}

POST /api/roadmap/generate
  Request:  {goal}
  Response: {beginner[], intermediate[], advanced[]}

Response Times:
  - ATS Analysis:   2-5 seconds
  - Skill Gap:      1-2 seconds
  - Roadmap:        <1 second
